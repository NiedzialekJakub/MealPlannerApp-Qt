MealPlanner (Qt / C++)

Desktopowa aplikacja ułatwiająca zarządzanie zawartością lodówki i wyszukiwanie pomysłów na posiłki. Projekt stworzony w C++ z wykorzystaniem frameworka Qt oraz bazy danych SQLite.

Główne funkcje
Wirtualna lodówka:Możliwość dodawania, usuwania i przeglądania posiadanych składników.
Inteligentne wyszukiwanie:Wyszukiwarka przepisów, która filtruje lokalną bazę danych na podstawie zaznaczonych składników.
Baza danych SQLite:Lokalne, szybkie przechowywanie wszystkich przepisów i składników.

Technologie
Język: C++
Framework: Qt (Qt Creator)
Baza danych:SQLite

Jak uruchomić projekt?
1. Sklonuj repozytorium na swój dysk:
   `git clone https://github.com/TwojaNazwa/MealPlannerApp-Qt.git`
2. Otwórz plik `CMakeLists.txt` w środowisku Qt Creator.
3. Skompiluj i uruchom projekt. Baza danych jak i pliki zasobów połączą się automatycznie dzięki ścieżkom względnym.

EN

MealPlanner (Qt / C++)

A desktop application designed to help you manage your fridge inventory and find meal ideas. The project is built in C++ using the Qt framework and an SQLite database.

Key Features
Virtual fridge:The ability to add, remove, and view your available ingredients.
Smart search: A recipe search engine that filters the local database based on selected ingredients.
SQLite database:Fast, local storage for all recipes and ingredients.

Technologies
Language: C++
Framework: Qt (Qt Creator)
Database: SQLite

How to run the project?
1. Clone the repository to your local drive:
   `git clone https://github.com/TwojaNazwa/MealPlannerApp-Qt.git`
2. Open the `CMakeLists.txt` file in Qt Creator.
3. Build and run the project. The database and resource files will connect automatically thanks to relative paths.
